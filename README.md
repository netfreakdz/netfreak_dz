# netfreak_dz

A fast, minimal, and modern personal website.

## Run locally
- Open `index.html` directly in your browser, or
- Serve the folder:
  - Python: `python -m http.server 3000`
  - Node: `npx serve .`

## Customize
- Update text and links in the HTML files.
- Edit styles in `assets/css/styles.css`.
- Update theme behavior in `assets/js/main.js`.
- Replace `favicon.ico` if desired.

## Deploy
- Any static host works: GitHub Pages, Netlify, Vercel.
- Upload the folder or connect your repo and set the publish directory to the project root.
